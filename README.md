# Video transcriber
Video transcriber that utilizes OpenAI Whisper & yt-dlp to download and transcribe a video's audio using your local machine. Requires python & ffmpeg

## Usage
`pip3 install -r requirements.txt`

`python3 transcribe.py [video_link1], [video_link2]...`

## Required environment variables
Need both client_id and client_secret from twitch dev console
